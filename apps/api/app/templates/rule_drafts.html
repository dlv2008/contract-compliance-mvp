制度来源文档

先保存制度或经验文档，系统会记录来源 hash 并切分为 chunks，后续 LLM 草稿生成必须引用这些来源。

生效路径

  1. 导入制度来源文档
  2. 确认 chunks 与来源 hash
  3. 基于 chunks 生成 draft 草稿
  4. 审核通过为 approved
  5. 发布为 active 资产
  6. 复制配置集并绑定 active 资产
  7. 发布配置集新版后用于合同上传
