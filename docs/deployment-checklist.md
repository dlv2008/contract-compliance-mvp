# 本地 WSL 演示检查清单

- [ ] `Ubuntu` WSL 实例可以正常启动

- [ ] 基础依赖已按你现有方式可用
- [ ] `ragflow_server.py` 已启动

- [ ] `RAGFlow` 前端或后端端口可以从 Windows 访问
