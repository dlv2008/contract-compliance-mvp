# RAGFlow 容量说明

截至 `2026-05-28`，已确认：

- GitHub 上观察到的最新稳定版本为 `v0.25.6`
- 官方 README 中的自托管前提包含 `RAM >= 16 GB`
- 当时云服务器可用内存约为 `7.3 GiB`

由此带来的实际判断是：

- 不应假设“同机部署 `RAGFlow`”在第一天就一定稳定
- `rag.trendbot.cn` 应视为后续阶段预留入口
- 合规审查应用应保持松耦合，确保它未来既可以接：
  - 后续自托管的 `RAGFlow`
  - 其他知识库服务
  - 或在 `MVP` 阶段直接采用“文件 + 规则”的兜底模式
