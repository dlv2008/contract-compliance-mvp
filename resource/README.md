# 财务合规 MVP 资源包

本目录是财务/合同合规审查 MVP 的最小资源集，目的不是做正式法务资料库，而是让 MVP 能立即开始：

- 知识库导入
- 合同抽取
- 规则判定
- 模型 A/B 评测
- 人工标注
- 报告演示

## 目录说明

- `01_合同样本`：用于抽取、规则和结果页演示的中文合同样本
- `02_制度样本`：用于知识库检索和制度依据引用的制度样本
- `03_标注与评测`：用于模型评测、人工标注和 gold label 保存
- `04_规则与字段`：字段口径和 MVP 规则清单
- `05_报告样本`：报告输出样例
- `06_UI与交互`：页面线框、信息架构和响应式布局说明

## 使用顺序

1. 先将 `02_制度样本` 导入 `RAGFlow`
2. 将 `01_合同样本` 作为测试合同上传
3. 用 `04_规则与字段` 配置字段 Schema 和规则 YAML
4. 用 `03_标注与评测` 跑 `MiniMax / Qwen 32B` 的 A/B 评测
5. 用 `05_报告样本` 对齐最终报告格式

## 资料来源说明

本资源包中的合同和制度文本，默认按 MVP 演示需要生成，避免直接复制外部长文本。

- 国家市场监督管理总局合同示范文本库：<https://htsfwb.samr.gov.cn/>
- 中国政府网关于全国合同示范文本库的介绍：<https://www.gov.cn/xinwen/2022-06/03/content_5693844.htm>
- 财政部会计司对《企业内部控制应用指引第6号——资金活动》的解读：<https://kjs.mof.gov.cn/zhengcejiedu/201005/t20100528_320204.htm>
- 财政部会计司对《企业内部控制应用指引第7号——采购业务》的解读：<https://kjs.mof.gov.cn/zhengcejiedu/201006/t20100611_322435.htm>

这些来源用于确定条款结构和控制点，不作为直接复制文本来源。
