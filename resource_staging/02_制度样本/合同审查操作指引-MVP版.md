# 合同审查操作指引（MVP版）

【POLICY-REV-001】审查目标  
合同审查应输出结论、依据、风险等级和整改建议，不得仅给出模糊意见。

【POLICY-REV-002】审查结论分级  
风险等级分为 `green`、`yellow`、`red`。其中 `red` 应进入人工复核。

【POLICY-REV-003】证据要求  
每条风险结论必须绑定至少一处合同证据和一处制度依据。

【POLICY-REV-004】自动续约审查  
出现自动续约、默认展期、默认验收、业务联系人确认优先等条款时，应重点审查审批链和授权边界。

【POLICY-REV-005】争议解决条款审查  
争议解决地不在甲方常用辖区，或约定对甲方明显不利的仲裁/诉讼安排，应标记为关注项。

【POLICY-REV-006】补充协议与变更控制  
合同变更必须通过正式书面协议完成。业务联系人邮件、IM 消息或口头确认，不得直接替代正式补充协议。
