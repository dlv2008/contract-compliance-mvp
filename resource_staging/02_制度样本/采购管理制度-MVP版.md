# 采购管理制度（MVP版）

【POLICY-PUR-001】采购合同必备条款  
采购合同应明确采购标的、数量、质量标准、交付时间、验收方式、付款条件、发票要求、售后服务和违约责任。

【POLICY-PUR-002】采购预付款控制  
设备与物资采购预付款原则上不得超过合同总价的 30%；超过比例的，应由采购负责人和财务负责人审批。

【POLICY-PUR-003】验收要求  
设备或物资到货后，应由需求部门、采购部门或指定验收人进行验收，不得仅以签收替代质量验收。

【POLICY-PUR-004】质量保证要求  
设备类采购应约定质保期限、保修责任、故障响应和更换维修责任。

【POLICY-PUR-005】收款账户一致性  
采购付款的收款账户应与签约主体一致；不一致的，原则上不得付款。

【POLICY-PUR-006】框架合同续签控制  
框架采购合同不得默认自动续约；确需延续的，应在期满前重新审批并补充签约。
